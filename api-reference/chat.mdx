### 基础参数

<ParamField body="model" type="string" required>
  要调用的模型名称。例如：`gpt-4o`、`claude-3-5-sonnet` 或 `deepseek-chat`。
</ParamField>

<ParamField body="messages" type="array" required>
  对话消息列表。
  <Expandable title="展开查看 Message 对象格式">
    <ParamField body="role" type="string" required>
      角色名称。可选值：`system`, `user`, `assistant`。
    </ParamField>
    <ParamField body="content" type="string" required>
      消息的具体内容。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="temperature" type="number" default="0.7">
  采样温度，介于 0 到 2 之间。较高的值会使输出更加随机。
</ParamField>

### 授权与鉴权

您需要在 HTTP 请求头 (Headers) 中传递您的 API Key：
<ParamField header="Authorization" type="string" required>
  格式为 `Bearer YOUR_API_KEY`
</ParamField>
