## 计费规则说明

<Info>
  **提示：** 我们的计费逻辑与官方完全对齐，不设任何隐藏溢价。所有充值金额永久有效。
</Info>

### 1. 余额扣除机制
每当您的 API Key 成功发起一次请求，系统会根据该模型的单价及消耗的 Token 数，实时从您的账户余额中扣除对应金额。

### 2. 免费额度
新注册用户将自动获得 **$0.50** 的免费试用额度，可用于测试所有支持的模型。

### 3. 高频率并发 (Tier)
默认情况下，所有模型支持 50 RPM (每分钟请求数)。如需更高并发，请联系技术支持申请提升等级。
